# Hydra++

This repository contains the source for the Hydra++ GitHub Pages website:
https://hydra-plusplus.github.io/

Hydra++ is a hierarchical 3D scene graph construction system with object-level
shape estimation. The current project page is based on the manuscript at
`static/pdfs/hydra-plusplus.pdf`.

Publication venue, arXiv link, video link, and final citation details are
currently marked as `(TBU)` in the website.

If you find Hydra++ useful for your work, please cite:

